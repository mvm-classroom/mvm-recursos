# Exercici: La nostra base de dades XML

Amb els coneixements i eines que coneixem a aquest punt del curs sobre XML (sintaxi bàsica, namespaces, DTD...) hem de crear una petita base de dades.

Farem servir [aquesta base de dades](https://github.com/mvm-classroom/mvm-recursos/blob/main/cicles/ASIX/GBD/cas-practic-1.md) com exemple.

## 1. Creació del DTD
Hem de crear un DTD per definir les regles que haurà de tenir el nostre format XML: 
- Quines etiquetes es poden fer servir
- Com es poden anidar
- En quin ordre
- Quins tipus de dades
- Atributs
- Referències

## 2. Creació del XML
Hem de crear un document XML on indiquem que estem fent servir el DTD del punt 1.
Heu d'afegir un mínim de 3 elements per cada entitat

## 3. Validació
Fent servir l'extensió de `Visual Studio` Code anomenada `XML Language Support by Red Hat` validarem que el nostre fitxer es coherent amb les regles que hem definit al DTD.
- Forceu alguns errors, de manera que algunes etiquetes i/o atributs no compleixin les regles del DTD
- Corregir els errors i deixar el document correcte.
